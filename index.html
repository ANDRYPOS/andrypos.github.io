Review Aplikasi:
Keterangan :
*) Aplikasi ini dibuat untuk memenuhi kebutuhan store dalam control absensi prakerin oleh administrator.

Aplikasi - aplikasi pendukung:
Inventory (Integrated Inventory System)
Keterangan :
*) Aplikasi ini dibuat untuk memenuhi tugas kuliah Pemrograman2.
*) Aplikasi ini dibuat dengan latar belakang memenuhi kebutuhan administrasi dalam melakukan control data inventory.
*) Aplikasi ini terintegrasi dalam bahasa pemrograman Java GUI NetBeans dan MySQL.
*) User melakukan login dan akan masuk kedalam menu utama.
*) Hak akses user dibagi menjadi tiga level (user, supervisor, administrator).
*) Level user hanya bisa melakukan pendataan barang saja.
*) Level supervisor bisa melakukan pendataan, report dan mengakses dashboard.
*) Level administrator merupakan level dengan full akses mulai dari mendaftarkan user pengguna sampai dengan mengakses dashboard.
Link aplikasi : github - ANDRYPOS/inventorys-java-netbeans
